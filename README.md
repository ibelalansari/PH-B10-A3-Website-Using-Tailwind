# Website Using Tailwind CSS

## Overview

This project was completed as Assignment 3 of the Programming Hero Batch 10 web development course.

The goal of this assignment was to build a fully responsive website using Tailwind CSS while learning a utility-first approach to frontend development.

## Technologies Used

- HTML5
- Tailwind CSS

## Features

- Responsive layout
- Mobile-friendly design
- Utility-first styling with Tailwind CSS
- Hero section
- Featured content
- Clean and modern interface
- Optimized page structure

## What I Learned

Through this project, I gained practical experience with:

- Tailwind CSS fundamentals
- Utility-first CSS workflow
- Responsive web design
- Layout customization
- Building reusable UI components
- Organizing frontend project structure
